gen-report: Generates an "End of Day Report" for my records.
    FLAGS:
        -h -- Displays this help text

        -n -- Set client name

        -p -- Set client phone number

        -e -- Set Client email address

        -b -- Set client bussiness name
    
    Example Usage:
        gen-report -h
        gen-report -n "Bob Ross" -p "(123)456-7890" -e "some@email.com" -b "Google Inc"
